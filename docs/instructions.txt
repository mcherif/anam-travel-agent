## What's Different from V1?

Simplified:
- No Ollama or local LLM
- No FastAPI backend
- No text-inference timing

Added:
- UIOrchestrator class (state machine + tool execution)
- DebugHUD for observability
- Mic test view for audio debugging

Same:
- Mapbox integration
- Anam SDK usage
- React frontend

## Architecture Layers

Presentation Layer
- TravelAgentDemo.tsx (coordinates everything)
- DebugHUD.tsx (shows metrics)

Orchestration Layer
- UIOrchestrator.ts (state machine + tool execution)

Data Layer
- landmarks_db.json (static data)
- landmarks.ts (typed exports)

External Services
- Anam SDK (persona + tools)
- Mapbox GL (mapping)
- Backend token server (session tokens)
